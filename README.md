# Galería Visual Dragon Ball

## Descripción del Proyecto
Este proyecto es una galería visual que muestra **20 personajes de Dragon Ball Z**, diseñada con HTML y CSS. Cada personaje tiene su propia tarjeta con imagen, nombre y descripción. La página incluye además:

- Un **título animado** con gradiente de colores.
- Una **galería de bolas de dragón** interactiva.
- Un **subtítulo de exploración** que invita al usuario a descubrir los personajes.
- **Footer con autor**.

El objetivo del proyecto es combinar diseño visual y estructura HTML/CSS para crear una experiencia interactiva y atractiva.

---

## Decisiones de Diseño

### Paleta de Colores
- **Degradado de fondo:** tonos amarillos y naranjas (`#f9f0d9`, `#ffe0b2`) para dar sensación de energía y vibrancia, inspirada en el estilo de Dragon Ball.  
- **Tarjetas de personajes:** gradientes claros (`#fff3e0`, `#ffd54f`) con borde naranja (`#ff9900`) para destacar cada personaje.  
- **Título animado:** gradiente multicolor que se mueve para generar dinamismo visual.

### Gradientes y Efectos
- El **título principal** usa un gradiente animado con varios colores del universo Dragon Ball para atraer la atención.  
- Las **tarjetas de personajes** tienen sombra y efecto hover (zoom y desplazamiento) para dar sensación de profundidad.  
- La **galería de bolas de dragón** tiene efecto hover con rotación y aumento de tamaño para interacción visual.

### Optimización de Imágenes
- Las imágenes se guardaron en formato `.webp` para reducir el tamaño y mantener calidad visual.  
- Todas las imágenes tienen dimensiones consistentes para que la galería se vea ordenada y responsiva.
